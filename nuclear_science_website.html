核工程技术科普

探索核能的奥秘，了解清洁能源的未来

清洁能源

核能是最清洁的大规模发电方式之一，几乎不产生温室气体排放。

安全可靠

现代核电站采用多重安全系统，确保运行安全和环境保护。

高效发电

核能具有极高的能量密度，一颗铀燃料丸相当于一吨煤的能量。

可持续发展

核燃料可以回收利用，第四代反应堆将大幅提高燃料利用率。

经济效益

核电具有稳定的发电成本和长期的经济效益。

全球发展

全球已有30多个国家运行着450多座核电机组。

反应堆堆型科普

商用堆型

压水堆 (PWR)

压水堆是目前世界上应用最广泛的反应堆类型，约占全球核电机组的65%。

  • • 使用轻水作为慢化剂和冷却剂
  • • 一回路水保持高压，防止沸腾
  • • 二回路产生蒸汽驱动汽轮机
  • • 安全性能优良，技术成熟

沸水堆 (BWR)

沸水堆允许冷却剂在反应堆内沸腾，产生的蒸汽直接驱动汽轮机。

  • • 简化的系统设计
  • • 只有一个冷却回路
  • • 较高的热效率
  • • 自然循环冷却

重水堆 (HWR)

重水堆使用重水作为慢化剂，可以使用天然铀作为燃料。

  • • 使用天然铀燃料
  • • 在线换料能力
  • • 优异的中子经济性
  • • 适合铀资源丰富的国家

石墨气冷堆 (GCR)

使用石墨作为慢化剂，二氧化碳作为冷却剂的反应堆类型。

  • • 高温运行能力
  • • 良好的安全特性
  • • 可在线换料
  • • 主要用于英国

先进堆型

高温气冷堆 (HTGR)

使用氦气冷却，石墨慢化，运行温度可达950°C。

  • • 固有安全性
  • • 高热效率
  • • 制氢能力

钠冷快堆 (SFR)

使用液态钠冷却的快中子反应堆，具有增殖能力。

  • • 燃料增殖
  • • 废料焚烧
  • • 高燃耗

熔盐堆 (MSR)

燃料溶解在熔融盐中，具有独特的安全特性。

  • • 低压运行
  • • 固有安全
  • • 在线处理

超临界水冷堆 (SCWR)

使用超临界水冷却，简化系统设计。

  • • 高热效率
  • • 简化系统
  • • 经济性好

超高温气冷堆 (VHTR)

运行温度超过1000°C的高温气冷堆。

  • • 超高温运行
  • • 高效制氢
  • • 工艺热应用
